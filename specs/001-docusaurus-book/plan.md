# Implementation Plan: Docusaurus Physical AI Textbook

**Branch**: `001-docusaurus-book` | **Date**: 2026-02-08 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/001-docusaurus-book/spec.md`

## Summary

Build a comprehensive Physical AI & Humanoid Robotics textbook as a Docusaurus 3.x static site with 16 chapters across 4 modules (ROS 2, Gazebo/Unity, NVIDIA Isaac, VLA Models) covering 13 weeks of coursework. The site will be deployed to GitHub Pages with auto-generated sidebar navigation, Mermaid diagrams, syntax-highlighted code blocks, dark mode, local search, and responsive mobile layout. Custom React component slots (PersonalizeButton, TranslateButton, ChatWidget) will be globally registered for future feature integration.

## Technical Context

**Language/Version**: JavaScript/Node.js 20 LTS, MDX v3
**Primary Dependencies**: Docusaurus 3.9.x, @docusaurus/theme-mermaid, docusaurus-search-local

**Performance Goals**: Landing page load < 3 seconds, Lighthouse accessibility 90+
**Constraints**: GitHub Pages hosting (static only), no server-side rendering
**Scale/Scope**: 16 chapters, ~30,000-50,000 words, 4 modules, 3 appendices
